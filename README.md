# СЭД ЖАК ЖАГУ — Электронный документооборот колледжа

Веб-система электронного документооборота для сотрудников Жалал-Абадского колледжа (ЖАК), структурного подразделения ЖАГУ, Кыргызская Республика.

Разработана для автоматизации внутреннего документооборота: создание приказов, служебных записок, отчётов; многоэтапное согласование; электронная подпись; учёт входящей корреспонденции.

## Стек

- **Next.js 15+** (App Router) + **TypeScript**
- **Tailwind CSS v4** (neumorphic design)
- **Prisma ORM v5** + **PostgreSQL**
- **Auth.js v5** (NextAuth, CredentialsProvider, JWT)
- **React Hook Form** + **Zod**
- **AES-256-GCM** + **SHA-256** (ЭП)

## Требования

- Node.js 18+
- PostgreSQL 14+

## Быстрый старт

```bash
# 1. Клонировать
git clone <repo>
cd <project>

# 2. Установить зависимости
npm install

# 3. Настроить окружение
cp .env.example .env
# Отредактировать: DATABASE_URL, NEXTAUTH_SECRET, ENCRYPTION_KEY

# 4. База данных
npx prisma generate
npx prisma db push
npx prisma db seed

# 5. Запустить разработку
npm run dev
```

|------------|----------|

## Ролевая модель

| Роль | Описание | Доступ |
|------|----------|--------|
| INITIATOR | Инициатор (преподаватель) | Создание документов, просмотр своих |
| VALIDATOR | Согласующий (зав.ПЦК, нач.уч.части) | Согласование, возврат на доработку |
| SIGNER | Подписант (директор, зам.директора) | Утверждение, ЭП |
| REGISTRAR | Регистратор (делопроизводитель) | Входящие, регистрация, архив |
| ADMIN | Администратор системы | Полный доступ, управление сотрудниками |

## Функции

- **Документооборот**: создание внутренних документов (приказы, распоряжения, протоколы, акты, служебные записки, договоры, отчёты)
- **Маршрутизация**: многоэтапное согласование по шаблонам (ПЦК → Учебный отдел → Зам.директора → Директор → Регистратор)
- **Электронная подпись**: AES-256-GCM шифрование SHA-256 хеша документа, визуальный штамп "УТВЕРЖДАЮ"
- **Версионность**: автоматическое сохранение каждой редакции с историей изменений
- **Входящая корреспонденция**: регистрация, резолюция директора, назначение исполнителя, контроль сроков
- **Уведомления**: о новых задачах, результатах согласования, просрочках
- **Аудит**: полное логирование всех действий с IP и user-agent
- **Поиск и архив**: фильтрация по типу, статусу, дате; поиск по тексту
- **PWA**: офлайн-режим, иконки, manifest
